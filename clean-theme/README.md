# Clean Theme - WordPress Theme

Чистая, универсальная тема для WordPress, готовая к наполнению контентом через штатные инструменты CMS.

## 📋 Описание

Clean Theme — это минималистичная тема без заранее прописанного контента, заглушек или неизменяемых текстовых блоков. Вся структура и функциональность полностью настраиваются через админ-панель WordPress.

## ✨ Возможности

### Поддержка WordPress Features
- `title-tag` — автоматическое управление заголовком страницы
- `post-thumbnails` — миниатюры записей и страниц
- `custom-logo` — загрузка логотипа через Настройщик
- `html5` — семантическая разметка HTML5
- `align-wide` и `responsive-embeds` — поддержка Gutenberg
- `widgets` — виджеты во всех областях
- `custom-header` и `custom-background` — кастомизация фона и шапки
- `block-styles` и `editor-styles` — стили для редактора блоков

### Области виджетов
- **Header Widgets** — виджеты в шапке (desktop)
- **Mobile Contact** — контакты в мобильном меню
- **Content Top** — над основным контентом (для слайдеров, баннеров)
- **Content Bottom** — под основным контентом
- **Sidebar** — боковая панель
- **Footer Left** — левая колонка футера
- **Footer Right** — правая колонка футера

### Меню навигации
- **Primary Menu** — главное меню (desktop + mobile)
- **Footer Menu** — меню в футере (опционально)

### Адаптивный дизайн
- Mobile-first подход
- Гамбургер-меню с плавной анимацией для мобильных устройств
- Responsive breakpoint: 1024px

## 🚀 Установка

1. Скопируйте папку `clean-theme` в `/wp-content/themes/`
2. Активируйте тему в админ-панели WordPress: **Внешний вид → Темы**
3. Настройте тему через **Внешний вид → Настроить**

## ⚙️ Настройка

### Через Настройщик тем (Внешний вид → Настроить)

#### Theme Options
- **Header Settings** — показать/скрыть виджеты в шапке
- **Colors** — акцентный цвет, цвет ссылок, цвет фона футера
- **Mobile Menu** — включить/выключить анимацию меню
- **Footer Settings** — показать меню в футере, текст копирайта
- **Typography** — базовый размер шрифта

### Через Виджеты (Внешний вид → Виджеты)
Добавляйте виджеты в зарегистрированные области:
- Телефон и соцсети в шапку
- Слайдеры и баннеры в Content Top/Bottom
- Любой контент в сайдбар и футер

### Через Меню (Внешний вид → Меню)
Создайте меню и назначьте их на локации:
- Primary Menu — главная навигация
- Footer Menu — навигация в футере

## 🎨 Стилизация

### CSS переменные для кастомизации
Тема использует чистый CSS без препроцессоров. Для изменения стилей:
1. Используйте дочернюю тему (recommended)
2. Добавьте custom CSS через **Внешний вид → Настроить → Дополнительные стили**
3. Отредактируйте `style.css` в дочерней теме

### Цвета по умолчанию
- Фон: `#ffffff`
- Текст: `#333`
- Акцент/ссылки: `#0073aa`
- Футер фон: `#f5f5f5`
